### Git으로 협업하기 

- `git pull`(변경사항 갱신)
- `git clone`(저장소 복제)
- `git fork`(저장소 요청)
- `git set -url`(원격 저장소 변경)
- `git reset`(커밋 되돌리기)

### Branch(분기)

- Branch란?
- `git checkout -b`(분기 생성)
- branch 원격 저장소에 전송
- `git checkout`(branch 변경)
- `git branch -d`(분기 삭제)
- `git merge`(분기 병합)
- branch clone(분기 복제)
